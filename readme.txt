基于Qt开发的图书管理系统具有以下功能：
1。注册用户管理（用户注册、用户登录、修改用户信息）
2。图书信息管理（添加新图书、修改图书信息、删除图书）
3。图书借阅管理（借书、还书、预定借阅图书、查询借阅信息）
4、图书管理员管理（查看全部图书列表、查看图书借阅列表、查看注册用户列表）
